# 🚀 ProjectLens – AI-Powered Project Evaluation Platform

> An intelligent system that evaluates developer projects from an interviewer’s perspective.

---

## 📌 Overview
ProjectLens helps students and developers understand how impressive their projects are in real-world hiring scenarios.

It analyzes projects and provides:
- Technical depth
- Code quality
- Real-world impact
- Resume readiness

---

## 🎯 Problem Statement
Students often build projects without knowing whether they are good enough for interviews.

Existing platforms do not provide interviewer-oriented evaluation, leading to confusion and missed opportunities.

---

## 💡 Solution
ProjectLens solves this by:
- Analyzing projects using AI
- Generating impressiveness scores
- Providing actionable feedback
- Suggesting resume improvements
- Creating interview questions

---

## ✨ Features

### 🔗 Project Input
- Upload GitHub repository link
- Extracts:
  - README
  - Key source files
  - Tech stack

---

### 🧠 AI Evaluation
- Impressiveness Score (0–100)
- Complexity Level
- Interviewer Verdict

---

### 📈 Feedback
- Strengths
- Weaknesses
- Improvement Suggestions

---

### 🧾 Resume Generator
- Generates ATS-friendly resume points

---

### 🎤 Interview Questions
- Project-based interview questions

---

### ⭐ Score Breakdown
- Innovation
- Code Quality
- UI/UX
- Real-world Impact

---

### 🧭 Improvement Roadmap
- Step-by-step improvement guidance

---

## 📏 Analysis Scope

### ✅ Analyzes
- README.md
- 3–5 important files
- Project structure

### ⚙️ Limits
- Max size: ~3MB
- Max files: 5

### 🚫 Ignores
- node_modules
- build folders
- large assets

---

## 🏗️ Tech Stack

**Frontend:** React.js, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**APIs:** OpenAI API, GitHub API  

---

## 🔄 Workflow

User Input → Data Extraction → Preprocessing → AI Analysis → Scoring → Dashboard
